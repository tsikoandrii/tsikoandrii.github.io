Stage of works

01 Formation of technical specifications

Where does it all start? Of course, from the formation of the technical specifications. Without this “document”, nothing will work. As a rule, most customers have those with them. the task. If you don’t have one, this is not a problem! We will create it by joint efforts.
What should be in thech. specification?
- layout type;
- wish for libraries, scripts, etc. (optional)
- deadlines;
Design is necessarily provided. (After all, we have not yet grown to turnkey sites :))

Stage of works

02 Layout

This is the main thing we do. This is a time consuming process. This is the conversion of a .psd, .ai file or Figma-design to a static layout. You can also decorate the static layout with different scripts (slider, adaptability, animations, etc.). All of this can be implemented using jQuery.
Layout happens:
- rubber;
- fixed;
- adaptive;
We recommend you: rubber layout + mobile (mobile version)
